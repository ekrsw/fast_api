# FastAPI APIサーバー

## 概要

このプロジェクトは、FastAPIを用いたシンプルなAPIサーバーです。SQLAlchemyでPostgreSQLと連携し、DockerおよびDocker Composeでコンテナ化されています。Nginxをリバースプロキシとして使用しています。
